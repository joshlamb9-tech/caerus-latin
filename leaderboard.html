The Ebbinghaus Forgetting Curve

Hermann Ebbinghaus (1885) showed that memory decays exponentially without review — we lose roughly 70% of new information within 24 hours if we don't revisit it. This leaderboard only counts words you have reviewed within the last 21 days. A word you haven't practised recently doesn't count — because if you haven't reviewed it, you're likely forgetting it.
