Extended schema adds two fields:
- `gcse_only`: boolean — true if this word appears in OCR J282 DVL but NOT in the CE list; false for CE carry-overs
- `principal_parts`: array of strings for verbs only, null for all other POS
  Example: `["amo", "amare", "amavi", "amatum"]`
  Irregular example: `["sum", "esse", "fui", "-"]`
  If a verb has no supine in common use: `["curro", "currere", "cucurri", "cursum"]`
