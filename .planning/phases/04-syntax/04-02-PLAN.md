function renderExercises(container, pool) {
  var lastIndex = -1;
  function showQuestion() {
    var idx;
    if (pool.length === 1) { idx = 0; }
    else { do { idx = Math.floor(Math.random() * pool.length); } while (idx === lastIndex); }
    lastIndex = idx;
    var q = pool[idx];
    var shuffled = q.options.slice().sort(function() { return Math.random() - 0.5; });
    var answered = false;
    container.innerHTML = '<div class="qcheck">'
      + '<div class="qcheck-header"><p class="qcheck-label">Quick check</p>'
      + '<span class="qcheck-count">' + (idx + 1) + ' of ' + pool.length + '</span></div>'
      + '<p class="qcheck-q">' + q.display + '</p>'
      + '<p style="font-size:0.85rem;color:#4a5568;font-style:italic;">' + q.translation + '</p>'
      + '<div class="qcheck-opts">'
      + shuffled.map(function(o) { return '<button class="qcheck-btn" data-val="' + o + '">' + o + '</button>'; }).join('')
      + '</div>'
      + '<div class="qcheck-fb" style="display:none"></div>'
      + '</div>';
    container.querySelectorAll('.qcheck-btn').forEach(function(btn) {
      btn.addEventListener('click', function() {
        if (answered) return;
        answered = true;
        var correct = btn.dataset.val === q.correct;
        container.querySelectorAll('.qcheck-btn').forEach(function(b) {
          b.disabled = true;
          if (b.dataset.val === q.correct) b.classList.add('qcheck-correct');
          else if (b === btn && !correct) b.classList.add('qcheck-wrong');
        });
        var fb = container.querySelector('.qcheck-fb');
        fb.style.display = 'block';
        fb.innerHTML = '<p class="' + (correct ? 'fb-correct' : 'fb-wrong') + '">'
          + (correct ? '\u2713 Correct \u2014 you knew that.' : 'Not yet \u2014 it\u2019s <strong>' + q.correct + '</strong>') + '</p>'
          + '<p class="fb-explanation">' + q.explanation + '</p>'
          + (pool.length > 1 ? '<button class="qcheck-next-btn">Try another \u2192</button>' : '');
        if (pool.length > 1) {
          fb.querySelector('.qcheck-next-btn').addEventListener('click', showQuestion);
        }
      });
    });
  }
  showQuestion();
}

**participles.html** (`gcse/syntax/participles.html`):
- Intro: "A participle is a verbal adjective — it comes from a verb but describes a noun. Latin uses three participles at GCSE: the present active, the perfect passive, and the future active."
- Explanation sections:
  - Present active participle: stem + -ns (gen. -ntis); agrees with noun in case/number/gender; translates as "verb-ing" or "while verb-ing". E.g. servus cantāns = the singing slave / the slave singing.
  - Perfect passive participle (PPP): 4th principal part; agrees in case/number/gender; translates as "having been verb-ed". E.g. captī = having been captured.
  - Future active participle: stem + -ūrus/-ūra/-ūrum; translates as "about to verb". E.g. reditūrus = about to return.
- Worked examples (3+):
  1. puer currentem canem spectābat. — The boy was watching the running dog. (present active participle)
  2. mīlitēs captī ad urbem ductī sunt. — The captured soldiers were led to the city. (PPP)
  3. rēx moritūrus populō locūtus est. — The king, about to die, spoke to the people. (future active participle)
  4. fēmina in forō stāns verba audīvit. — The woman standing in the forum heard the words.
- Exam tip: "Check that the participle agrees in case, number, and gender with the noun it describes — not with the subject of the main verb. A participial phrase almost always sits immediately next to the noun it modifies."
- See also: ablative-absolute.html, indirect-statement.html
- Exercise fetch: `fetch('../../data/exercises/gcse-participles.json')`

**ablative-absolute.html** (`gcse/syntax/ablative-absolute.html`):
- Intro: "The ablative absolute is a self-contained ablative phrase — a noun (or pronoun) and a participle both in the ablative — that is grammatically independent of the main clause."
- Explanation:
  - Both noun and participle are in the ablative case and agree in case (and number/gender if the participle is a PPP).
  - The phrase is grammatically separate from the main clause — the noun in the phrase is never the subject or object of the main verb.
  - Translation: "with X having been verb-ed", "when/since/after X verb-ed" — choose whichever fits the context.
  - Note: Latin has no present passive participle, so ablative absolutes often use a noun + noun or noun + adjective (e.g. Caesare consule = when Caesar was consul).
- Worked examples (3+):
  1. rege occīsō, mīlitēs fūgērunt. — When the king had been killed, the soldiers fled. (PPP absolute)
  2. duce profectō, impedīmenta relicta sunt. — After the general had set out, the baggage was left behind.
  3. hīs rēbus audītīs, omnēs timēbant. — When these things had been heard, everyone was afraid.
  4. Caesare duce, Rōmānī vīcērunt. — With Caesar as leader / When Caesar was leader, the Romans won. (noun + noun)
- Comparison table: title "Ablative Absolute vs Participial Phrase"; columns: Feature | Ablative Absolute | Participial Phrase; rows: Case of noun (Ablative | Nominative/Accusative matching main clause noun), Grammatical link to main clause (None — independent | Directly modifies a main clause noun), Typical translation (When/since/after/with | While/having been).
- Exam tip: "If you spot a noun and a participle both in the ablative with no grammatical connection to the main verb, that is your ablative absolute. The key test: can the ablative noun be the subject or object of the main verb? If not, it is ablative absolute."
- See also: participles.html, temporal-causal-concessive.html
- Exercise fetch: `fetch('../../data/exercises/gcse-ablative-absolute.json')`

**indirect-statement.html** (`gcse/syntax/indirect-statement.html`):
- Intro: "Indirect statement reports what someone said, thought, or believed. Latin uses the accusative + infinitive construction: the subject of the reported action goes into the accusative, and the verb becomes an infinitive."
- Explanation:
  - Trigger verbs (say, think, know, perceive, hear, see): dīcō, inquit, crēdō, putō, sciō, sentiō, audiō, videō.
  - The reported subject → accusative; the reported verb → infinitive.
  - Tense of infinitive: present infinitive = action at same time as main verb; perfect infinitive = action before main verb; future infinitive = action after main verb.
  - Latin future infinitive: future active participle + esse.
- Worked examples (3+):
  1. dīcō puerōs in forō esse. — I say that the boys are in the forum. (present infinitive: same time)
  2. crēdō mīlitēs fortiter pugnāvisse. — I believe that the soldiers fought bravely. (perfect infinitive: before)
  3. putābam rēgem reditūrum esse. — I thought that the king would return. (future infinitive: after)
  4. sciō tē errāre. — I know that you are making a mistake.
- Comparison table (title "The Three Indirect Constructions"):

**indirect-commands.html** (`gcse/syntax/indirect-commands.html`):
- Intro: "An indirect command reports an order or request. After verbs of ordering, asking, and persuading, Latin uses ut (positive) or nē (negative) + subjunctive."
- Explanation:
  - Trigger verbs: rogō, imperō, hortō/hortor, persuādeō, moneō, ōrō.
  - Positive command: ut + subjunctive — "to/that [subject] should verb"
  - Negative command: nē + subjunctive — "not to/that [subject] should not verb"
  - Exception: iubeō (order) takes accusative + infinitive (not ut + subjunctive) — same pattern as indirect statement.
  - The tense of the subjunctive follows sequence of tenses (link to sequence-of-tenses.html).
- Worked examples (3+):
  1. imperō mīlitibus ut discēdant. — I order the soldiers to leave. (present subjunctive: primary sequence)
  2. rogāvī amīcum nē id faceret. — I asked my friend not to do that. (imperfect subjunctive: secondary sequence)
  3. hortātus est nōs ut fortiter pugnārēmus. — He encouraged us to fight bravely.
  4. iussit mīlitēs discēdere. — He ordered the soldiers to leave. (iubeō + infinitive)
- Exam tip: "nē after a verb of ordering means 'not to' (negative indirect command). Don't confuse it with nē after a verb of fearing, which means 'that' (positive fear). The trigger verb is your first clue: ordering verbs → indirect command; fearing verbs → fearing clause."

- Intro: "An indirect question reports what someone asked or wanted to know. A question word (quis, quid, cur, ubi, quōmodo, num, utrum...an) introduces a clause with a subjunctive verb."
- Explanation:

  3. quaesīvit num mīlitēs iam advēnissent. — He asked whether the soldiers had already arrived. (num + pluperfect subjunctive)
  4. mīror quōmodo hoc fēceris. — I wonder how you did this.
