  <h1>4th &amp; 5th Declension Nouns</h1>
  <p class="page-intro">The 4th and 5th declensions are rarer than the first three but appear regularly in GCSE texts. Learn the key endings and you will recognise them instantly.</p>

  <!-- Section 1: 4th Declension -->
  <section id="fourth-declension">
    <h2>Fourth Declension</h2>
    <p>4th declension nouns are mostly masculine and end in <strong>-us</strong> in the nominative singular. The stem vowel is <strong>-u-</strong>, which appears in most endings.</p>
    <h3>Formation rules</h3>
    <ul>
      <li>Nominative singular: <strong>-us</strong> (e.g. <em>manus</em> = hand)</li>
      <li>Genitive singular: <strong>-ūs</strong> (long u — do not confuse with the nominative)</li>
      <li>Dative singular: <strong>-uī</strong></li>
      <li>Accusative singular: <strong>-um</strong></li>
      <li>Ablative singular: <strong>-ū</strong> (long u)</li>
      <li>Nominative/accusative plural: <strong>-ūs</strong></li>
      <li>Dative/ablative plural: <strong>-ibus</strong> (same as 3rd declension!)</li>
    </ul>
    <h3>Worked examples</h3>
    <p><em>Mīlitēs manibus pugnābant.</em> — The soldiers were fighting with their hands. (<em>manibus</em> = ablative plural, means)</p>
    <p><em>Fīlius manūs patris tenēbat.</em> — The son was holding the hands of his father. (<em>manūs</em> = accusative plural)</p>
    <div id="table-4th"></div>
  </section>

  <!-- Section 2: 5th Declension -->
  <section id="fifth-declension">
    <h2>Fifth Declension</h2>
    <p>5th declension nouns are mostly feminine and end in <strong>-ēs</strong> in the nominative singular. The most common is <em>rēs</em> (thing, matter) — it appears constantly in Latin texts.</p>
    <h3>Formation rules</h3>
    <ul>
      <li>Nominative/vocative singular: <strong>-ēs</strong></li>
      <li>Accusative singular: <strong>-em</strong></li>
      <li>Genitive/dative singular: <strong>-eī</strong></li>
      <li>Ablative singular: <strong>-ē</strong></li>
      <li>Nominative/vocative/accusative plural: <strong>-ēs</strong> (all three the same!)</li>
      <li>Genitive plural: <strong>-ērum</strong></li>
      <li>Dative/ablative plural: <strong>-ēbus</strong></li>
    </ul>
    <h3>Worked examples</h3>
    <p><em>Senex dē rē cogitābat.</em> — The old man was thinking about the matter. (<em>rē</em> = ablative singular after dē)</p>
    <p><em>Multās rēs in forō vīdimus.</em> — We saw many things in the forum. (<em>rēs</em> = accusative plural)</p>
    <div id="table-5th"></div>
  </section>

Use context: if it follows a noun as a possessive it is genitive; if it is the subject of the verb it is nominative. For the 5th declension, watch for <em>reī</em> (genitive/dative) — two syllables, long -ī.

**Bottom script (inline, after nav-toggle script):**
```js
(function () {
  // Fetch paradigm data and render tables
  fetch('../../data/grammar/gcse-nouns.json')
    .then(function(r) { return r.json(); })
    .then(function(data) {
      data.declensions.forEach(function(dec) {
        var targetId = dec.declension === 4 ? 'table-4th' : 'table-5th';
        var container = document.getElementById(targetId);
        if (!container) return;
        var cases = dec.cases;
        var html = '<table class="grammar-table">'
          + '<thead><tr><th>Case</th><th>Singular</th><th>Plural</th></tr></thead><tbody>';
        cases.forEach(function(c) {
          html += '<tr><td>' + c.charAt(0).toUpperCase() + c.slice(1) + '</td>'
            + '<td><em>' + dec.singular[c] + '</em></td>'
            + '<td><em>' + dec.plural[c] + '</em></td></tr>';
        });
        html += '</tbody></table>';
        if (dec.note) {
          html += '<p style="font-size:0.85rem;color:#4a5568;margin-top:-0.5rem;">' + dec.note + '</p>';
        }
        container.innerHTML = html;
      });
    });

  // Fetch exercises and render MCQ
  fetch('../../data/exercises/gcse-declensions.json')
    .then(function(r) { return r.json(); })
    .then(function(questions) {
      var container = document.getElementById('exercise-container');
      if (!container) return;
      renderExercises(container, questions);
    });

  function renderExercises(container, pool) {
    var lastIndex = -1;

    function showQuestion() {
      var idx;
      if (pool.length === 1) { idx = 0; }
      else { do { idx = Math.floor(Math.random() * pool.length); } while (idx === lastIndex); }
      lastIndex = idx;
      var q = pool[idx];
      var shuffled = q.options.slice().sort(function() { return Math.random() - 0.5; });
      var answered = false;

      container.innerHTML = '<div class="qcheck">'
        + '<div class="qcheck-header"><p class="qcheck-label">Quick check</p>'
        + '<span class="qcheck-count">' + (idx + 1) + ' of ' + pool.length + '</span></div>'
        + '<p class="qcheck-q">' + q.display + '</p>'
        + '<p style="font-size:0.85rem;color:#4a5568;font-style:italic;">' + q.translation + '</p>'
        + '<div class="qcheck-opts">'
        + shuffled.map(function(o) { return '<button class="qcheck-btn" data-val="' + o + '">' + o + '</button>'; }).join('')
        + '</div>'
        + '<div class="qcheck-fb" style="display:none"></div>'
        + '</div>';

      container.querySelectorAll('.qcheck-btn').forEach(function(btn) {
        btn.addEventListener('click', function() {
          if (answered) return;
          answered = true;
          var correct = btn.dataset.val === q.correct;
          container.querySelectorAll('.qcheck-btn').forEach(function(b) {
            b.disabled = true;
            if (b.dataset.val === q.correct) b.classList.add('qcheck-correct');
            else if (b === btn && !correct) b.classList.add('qcheck-wrong');
          });
          var fb = container.querySelector('.qcheck-fb');
          fb.style.display = 'block';
          fb.innerHTML = '<p class="' + (correct ? 'fb-correct' : 'fb-wrong') + '">'
            + (correct ? '\u2713 Correct \u2014 you knew that.' : 'Not yet \u2014 it\u2019s <strong>' + q.correct + '</strong>') + '</p>'
            + '<p class="fb-explanation">' + q.explanation + '</p>'
            + (pool.length > 1 ? '<button class="qcheck-next-btn">Try another \u2192</button>' : '');
          if (pool.length > 1) {
            fb.querySelector('.qcheck-next-btn').addEventListener('click', showQuestion);
          }
        });
      });
    }

    showQuestion();
  }
})();
```

  <h1>Passive Voice</h1>
  <p class="page-intro">In the passive voice, the subject receives the action rather than performing it. GCSE Latin requires the full passive for all tenses, including the compound perfect and pluperfect.</p>

  <section id="passive-formation">
    <h2>How the Passive is Formed</h2>
    <p>For the present, imperfect, and future, passive endings replace the active endings on the same stem:</p>
    <table class="grammar-table">
      <thead><tr><th>Person</th><th>Active ending</th><th>Passive ending</th></tr></thead>
      <tbody>
        <tr><td>1st sg.</td><td>-ō / -m</td><td>-or / -r</td></tr>
        <tr><td>2nd sg.</td><td>-s</td><td>-ris</td></tr>
        <tr><td>3rd sg.</td><td>-t</td><td>-tur</td></tr>
        <tr><td>1st pl.</td><td>-mus</td><td>-mur</td></tr>
        <tr><td>2nd pl.</td><td>-tis</td><td>-minī</td></tr>
        <tr><td>3rd pl.</td><td>-nt</td><td>-ntur</td></tr>
      </tbody>
    </table>
    <p>The <strong>perfect and pluperfect passive</strong> are compound tenses: perfect passive participle (PPP) + the appropriate form of <em>sum</em>. The PPP agrees with the subject in case, number, and gender.</p>
    <h3>The agent construction</h3>
    <p>The person <em>by whom</em> the action is done is expressed with <strong>ā/ab + ablative</strong>:</p>
    <p><em>Puella ā magistrō laudātur.</em> — The girl is praised <strong>by the teacher</strong>. (<em>ā magistrō</em> = by the teacher)</p>
    <p><em>Servī ā dominō vocābantur.</em> — The slaves were being called <strong>by the master</strong>. (<em>ā dominō</em> = by the master)</p>

  </section>
