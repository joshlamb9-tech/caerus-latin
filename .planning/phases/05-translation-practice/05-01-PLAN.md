10 sets, each with exactly 3 sentences. Topics (one per set):
1. Passive voice — perfect and imperfect passive forms
2. Ablative absolute — participial ablative phrases

4. Result clauses — tam/ita...ut + subjunctive
5. Indirect statement — accusative + infinitive
6. Indirect commands — imperō/iubeō + ut/nē + subjunctive
7. Temporal clauses — cum/postquam/ubi + appropriate tense
8. Conditionals — sī + indicative and sī + subjunctive
9. Relative clauses — quī/quae/quod agreement
10. Mixed constructions — combination of 2-3 constructions per sentence

Requirements:
- All Latin answers use macron accents throughout
- mark_scheme_note explains the KEY construction the examiner looks for (e.g. "requires purpose clause with ut + subjunctive")
- Sentences use vocabulary from the J282 DVL where possible
- Difficulty progresses within each set (sentence 1 simpler, sentence 3 harder)
- Mixed set (10) combines constructions from earlier sets
